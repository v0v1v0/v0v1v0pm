supportedFileFormats R Documentation

Supported file formats

Description

This function prints all file formats supported by MALDIquantForeign-package.

Usage

supportedFileFormats()

Details

Import

txt importTxt
tab importTab
csv importCsv
fid importBrukerFlex
ciphergen importCiphergenXml
mzXML importMzXml
mzML importMzMl
imzML importImzMl
analyze importAnalyze
cdf importCdf
msd importMsd

Export

tab exportTab
csv exportCsv
imzML exportImzMl
msd exportMsd
mzML exportMzMl

Value

a list with two named elements (import and export) containing a character vector of supported file types.

Author(s)

Sebastian Gibb

References

https://strimmerlab.github.io/software/maldiquant/

See Also

export, import

Examples

library("MALDIquantForeign")

supportedFileFormats()
