dat.sim2 R Documentation

Simulated psychotherapy treatment studies for 'outcome two'

Description

This data set includes 1000 simulated psychotherapy treatment studies. The population standardized mean difference for outcome two is d=0.80. The data set also includes two correlated study-level variables (i.e., moderators): treatment duration (dose) and average participant distress (stress).

This data accompanies a tutorial for meta-analysis:

Del Re, A. C. (2015). A Practical Tutorial on Conducting Meta-Analysis in R. The Quantitative Methods for Psychology.
