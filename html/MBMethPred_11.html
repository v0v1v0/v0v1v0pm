ModelMetrics R Documentation

Model metrics

Description

A function to extract the confusion matrix information.

Usage

ModelMetrics(Model)

Arguments

Model

A trained model.

Value

A list

Examples

xgboost <- XGBoostModel(SplitRatio = 0.2,
                        CV = 2,
                        NCores = 1,
                        NewData = NULL)
ModelMetrics(Model = xgboost)
