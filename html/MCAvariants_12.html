printwithaxes R Documentation

Secondary printing function

Description

The function is called from the main print function printmcacorporateris. It adds the names to objects.

Usage

printwithaxes(res, thenames)

Arguments

res

An R object.
thenames

A character vector of up to the same length as x.

Note

It is called from printmcacorporateris.

Author(s)

Rosaria Lombardo
