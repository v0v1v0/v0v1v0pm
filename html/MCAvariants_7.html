miocount R Documentation

The counting function

Description

The function that counts the number of individuals in each clusters automatically generated in ordered multiple correpsondence analysis.

Usage

miocount(x)

Arguments

x

The coordinates of axes

Note

This function is used in the function omcabasic when in the main function MCAvariants the input parameter is catype="omca".

Author(s)

Rosaria Lombardo
