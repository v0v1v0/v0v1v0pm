mrb R Documentation

Mean Relative Bias

Description

Calculate the MRB of an array of estimates relative to the true value.

Usage

mrb(rhohat, rho)

Arguments

rhohat

an array of estimators of rho.
rho

the true value of rho.

Value

the mean relative bias of rhohat array.

See Also

mb rmse

Examples

rho = 0.5
rhohat = 0.5 + rnorm(10)
mrb(rhohat,rho)
