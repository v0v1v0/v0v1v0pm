dphixy R Documentation

Scaled Bivariate Normal Density

Description

Bivariate normal density with mean 0 variance 1.

Usage

dphixy(x, y, rho)

Arguments

x, y

points value.
rho

correlation coefficient.

Value

the density value.

Examples

library(mvtnorm)
dmvnorm(c(1,-1),sigma = matrix(c(1,0.5,0.5,1),2,2))
dphixy(1,-1,0.5)
