geom R Documentation

Geometric mean

Description

Computes the geometric mean of a numeric vector.

Usage

geom(x)

Arguments

x

A numeric vector

Value

The value of the geometric mean.

Author(s)

Thomas Wieland

Examples

numvec <- c(10,15,20,25,30)  
# Creates a numeric vector "numvec"
mean(numvec)  
# Mean of numvec
geom(numvec)  
# Geometric mean of numvec
