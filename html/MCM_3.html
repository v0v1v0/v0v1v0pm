sim_datlmer R Documentation

Simulated Data Studying Social Mobility (Longitudinal)

Description

This is a simulated data used to study social mobility under longitudinal setting.

Usage

data("sim_datlmer")

Format

A data frame with 50000 observations on the following 14 variables.

id

an ordered factor with levels

obs

a numeric vector

eij

a numeric vector

origin

a factor with levels 1 2 3

destination

a factor with levels 1 2 3

origin_1

a numeric vector

origin_2

a numeric vector

origin_3

a numeric vector

destination_1

a numeric vector

destination_2

a numeric vector

destination_3

a numeric vector

yij

a numeric vector

age

a numeric vector

dir

a factor with levels 0 1 2

Examples

data(sim_datlmer)
