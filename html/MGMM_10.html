GenImputation R Documentation

Generate Imputation

Description

Generates a stochastic imputation of a data set from a fitted data set.

Usage

GenImputation(fit)

Arguments

fit

Fitted model.

Value

Numeric matrix with missing values imputed.

Examples

set.seed(100)

# Generate data and introduce missingness.
data <- rGMM(n = 25, d = 2, k = 1)
data[1, 1] <- NA
data[2, 2] <- NA
data[3, ] <- NA 

# Fit GMM.
fit <- FitGMM(data)

# Generate imputation.
imputed <- GenImputation(fit)
