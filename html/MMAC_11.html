EngineRPM R Documentation

RPM of Different Engines

Description

A dataset containing the revolutions per minute (RPM) and mass of different engines.

Usage

data(EngineRPM)

Format

A data frame with 39 observations on the following variables.

  • Mass in kilograms

  • RPM

Details

This dataset contains the revolutions per minute (RPM) and mass of different engines.

Source

Revolutions per minute of engines as a function of engine mass from pages 60-61. McMahon, Thomas A., and John Tyler Bonner. On Size and Life. New York: Scientific American Library, 1983.

Examples

data(EngineRPM)
