singt_data R Documentation

Single Sample t Example Data

Description

A simulated dataset for use in d.single.t and d.single.t.t, including gifted/honors student SAT scores from a specific school to use for comparison with the national average SAT score (1080) of gifted/honors students nationwide.

Usage

data(singt_data)

Format

A data frame including a single sample consisting of SAT scores of students from a gifted/honors program at a specific school.
