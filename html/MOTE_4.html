chisq_data R Documentation

Chi-Square Example Data

Description

Dataset for use in v.chi.sq, Individuals were polled and asked to report their number of friends (low, medium, high) and number of kids (1, 2, 3+) to determine if there was a relationship between friend groups and number of children. It was hypothesized that those with more children may have less time for friendship maintaining activities.

Usage

data(chisq_data)

Format

A data frame of number of friends and number of children

friends: number of reported friends kids: number of children
