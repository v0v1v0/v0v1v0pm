Table B22 - Baseball Data

Description

The table.b22 data frame has 30 observations on 12 variables.

Usage

numeric vector

numeric vector

numeric vector

numeric vector

numeric vector

numeric vector

ERA

numeric vector

SO

numeric vector

numeric vector

numeric vector

Errors

numeric vector
