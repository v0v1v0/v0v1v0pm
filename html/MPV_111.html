table.b25 R Documentation

Table B25 Golf Data

Description

The table.b25 data frame has 50 observations on 6 variables.

Usage

data(table.b25)

Format

This data frame contains the following columns:

Player

character vector

Average.Score

numeric vector

SG..Off.the.Tee

numeric vector

SG..Approach.to.Green

numeric vector

SG..Around.the.Green

numeric vector

SG..Putting

numeric vector

Source

Montgomery, D.C., Peck, E.A., and Vining, C.G. (2021) Introduction to Linear Regression Analysis. 6th Edition, John Wiley and Sons.

Examples

pairs(table.b25[,-1])
