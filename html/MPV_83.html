PRESS R Documentation

PRESS statistic

Description

Computation of Allen's PRESS statistic for an lm object.

Usage

PRESS(x)

Arguments

x

An lm object

Value

Allen's PRESS statistic.

Author(s)

W.J. Braun

See Also

lm

Examples

data(p4.18)
attach(p4.18)
y.lm <- lm(y ~ x1 + I(x1^2))
PRESS(y.lm)
detach(p4.18)
