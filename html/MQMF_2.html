%ni% R Documentation

'%ni%' identifies which element in x is NOT in y

Description

'%ni%' identifies which element in x is NOT in y

Usage

x %ni% y

Arguments

x

a vector of elements which can be numeric or character
y

a vector of elements which can be numeric or character

Examples

   x <- 1:10
   y <- 6:15
   x
   y
   x[(x %ni% y)]   # are not in y
