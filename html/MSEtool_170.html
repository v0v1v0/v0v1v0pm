makeMOM R Documentation

Utility for making multi-OMs

Description

Converts an OM to a single stock, single fleet MOM.

Usage

makeMOM(..., silent = FALSE)

Arguments

...

An OM.
silent

Should messages be printed out to the console?

Value

A class MOM object.

Author(s)

Q. Huynh

Examples

MOM <- makeMOM(testOM)
