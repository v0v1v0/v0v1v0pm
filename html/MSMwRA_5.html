Sex, Education and Reading Data

Description

The data set is used in Sample 15.2 and given in Table 15.1. The data set consists of 3 variables and 21 observations. This data is imaginary.

Usage

SER

Format

A data frame with 21 rows and 3 variables. The variables are as follows:

Sex

The sex of observations

Education

The education level of observations

Reading

The reading case of observations

Source

The data set is used in the book entitled Multivariate Statistical Methods with R Applications (R Uygulamalari ile Cok Degiskenli Istatistiksel Yontemler), H.Bulut (2018).
