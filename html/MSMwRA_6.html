Simulated Data 1

Description

The data set is used in Sample 3.1. The data set consists of 3 variables and 10 observations. This data is imaginary.

Usage

Sim1

Format

A data frame with 10 rows and 3 variables.

Source

The data set is used in the book entitled Multivariate Statistical Methods with R Applications (R Uygulamalari ile Cok Degiskenli Istatistiksel Yontemler), H.Bulut (2018).
