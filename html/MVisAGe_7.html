gene.annot R Documentation

Gene annotation data (hg38)

Description

Genomic position and cytoband annotation data.

Usage

gene.annot

Format

A matrix with 26417 rows and 3 columns

chr

chromosome containing the gene of interest

pos

genomic position (base pairs) for the gene of interest

cytoband

cytoband containing the gene of interest

Row names

gene symbols

Source

https://usegalaxy.org/
