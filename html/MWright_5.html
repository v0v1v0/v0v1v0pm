int_est2 R Documentation

Interval estimation for two-sided M-Wright distribution

Description

Confidence intervals for the model parameters.

Usage

int_est2(x, lev)

Arguments

x

numeric vector
lev

confidence level.

Value

matrix

Examples


mwright_2sided <- rmwright2(1000, 0.7, 0.4)
int_est2(mwright_2sided ,0.95)
