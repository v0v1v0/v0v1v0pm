as_line R Documentation

Layout as a line

Description

Layout as a line

Usage

as_line(angle = 0)

Arguments

angle

anticlockwise rotation angle

Value

A two-column matrix, each row giving the coordinates of a vertex, according to the ids of the vertex ids.

See Also

Other layout: as_arc(), as_circle_tree(), as_polyarc(), as_polycircle(), as_polygon(), c_net_layout()

Examples

as_line()(co_net)
c_net_plot(co_net, coors = as_line(pi / 2))
