c_net_from_edgelist R Documentation

Construct a network from edge_list dataframe

Description

Construct a network from edge_list dataframe

Usage

c_net_from_edgelist(
  edgelist,
  vertex = NULL,
  direct = FALSE,
  e_type = NULL,
  e_class = NULL
)

Arguments

edgelist

first is source, second is target, others are annotation
vertex

vertex metadata
direct

logical
e_type

set e_type
e_class

set e_class

Value

metanet

See Also

Other build: c_net_build(), c_net_set(), c_net_update(), multi_net_build()

Examples

data(edgelist)
edge_net <- c_net_from_edgelist(arc_count, vertex = arc_taxonomy)
edge_net <- c_net_set(edge_net, vertex_class = "Phylum", edge_width = "n")
c_net_plot(edge_net)
