homology_searched R Documentation

An example sequence information file.

Description

A dataset containing the information of reads for homology search.

Usage

homology_searched

Format

A list with 7 factors, each contains 1508 variables:

sample_name

sample name

Chr

altered chromosome

Pos

altered position

Ref

reference base

Alt

altered base

Direction

5' (pre) or 3' (post) sequence of the mutated base

Seq

sequence for homology search
