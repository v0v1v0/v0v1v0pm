perms R Documentation

Permutations

Description

Returns all possible permutations given the number of elements.

Usage

perms(n)

Arguments

n

Number of elements.

Value

Returns a matrix containing all possible permutations.

Author(s)

Volodymyr Melnykov, Wei-Chen Chen, and Ranjan Maitra.
