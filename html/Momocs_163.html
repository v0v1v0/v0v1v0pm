coo_nb R Documentation

Counts coordinates

Description

Returns the number of coordinates, for a single shape or a Coo object

Usage

coo_nb(coo)

Arguments

coo

matrix of ⁠(x; y)⁠ coordinates or any Coo object.

Value

either a single numeric or a vector of numeric
