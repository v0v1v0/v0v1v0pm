plot_silhouette R Documentation

Silhouette plot

Description

Only used, so far, after KMEDOIDS.

Usage

plot_silhouette(x, palette = pal_qual)

Arguments

x

object returned by KMEDOIDS
palette

one of palettes

Value

a ggplot plot

Examples


olea %>% opoly(5) %>%
    KMEDOIDS(4) %>%
    plot_silhouette(pal_qual_solarized)
