Smoking habits

Description

Frequency table representing smoking habits of different employees in a company

Usage

data(smoking)

Format

A data frame with 5 observations on the following 4 variables.

None

a numeric vector

Light

a numeric vector

Medium

a numeric vector

Heavy

a numeric vector

Details

Frequency table representing smoking habits of different employees in a company
