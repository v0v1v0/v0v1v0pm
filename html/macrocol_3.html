get.IBR R Documentation

IBR extraction

Description

Function to extract the nominal IBR rate time serie from the Colombian Central Bank.

Usage

get.IBR(nom = "ON")

Arguments

nom

The interest rate period.
The periods available are:

  • ON: Overnight

  • 1M: 1 Month

  • 3M: 3 Months

  • 6M: 6 Months

Default is "ON".

Value

The dataframe with dates and the IBR.

Examples

## Not run: 
ON <- get.IBR("ON");
M1 <- get.IBR("1M");
M3 <- get.IBR("3M");
M6 <- get.IBR("6M");

## End(Not run)
