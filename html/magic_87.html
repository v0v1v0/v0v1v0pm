magic.4n R Documentation

Magic squares of order 4n

Description

Produces an associative magic square of order 4n using the delta-x method.

Usage

magic.4n(m)

Arguments

m

Order n of the square is given by n=4m

Author(s)

Robin K. S. Hankin

See Also

magic

Examples

magic.4n(4)
is.magic(magic.4n(5))
