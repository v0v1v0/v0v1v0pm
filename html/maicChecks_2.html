eIPD R Documentation

Example ipd set

Description

Example ipd data

Usage

data("eIPD")

Format

A data frame with 42 observations on the following 2 variables. The illustrative example used in the reference manuscript (Glimm & Yau (2021)).

y1

a numeric vector

y2

a numeric vector

References

Glimm & Yau (2021). Geometric approaches to assessing the numerical feasibility for conducting matching-adjusted indirect comparisons. arXiv 2108.01896.
