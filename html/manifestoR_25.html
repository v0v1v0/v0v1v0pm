iff R Documentation

Apply a function if and only if test is TRUE

Description

otherwise return input value unchanged

Usage

iff(obj, test, fun, ...)

iffn(obj, test, fun, ...)

Arguments

obj

object to apply test and fun to
test

logical or function to apply to test
fun

function to apply
...

passed on to test

Details

iffn is ... if and only if test is FALSE
