Filter a dataset within a mrgmod

Description

Filter a dataset within a mrgmod

Usage

## S3 method for class 'mrgmod'
filter(.data, ..., .preserve = FALSE)

Arguments

.data

a mrgmod
..., .preserve

additional arguments for dplyr::filter()

Value

a mrgmod

Examples

library(magrittr)
mod <- mrgsolve::mcode("mod", "$CMT FOO", compile = FALSE)
mod %>%
  adm_rows(amt = c(100, 200, 300), cmt = 1) %>%
  filter(amt != 200) %>%
  get_data()
