%>% R Documentation

Pipe

Description

Uses the pipe operator (%>%) to chain statements. Useful for adding layers to a mapdeck map

Arguments

lhs, rhs

A mapdeck map and a layer to add to it

Examples



token <- "your_api_token"
mapdeck(token = token) %>%
add_scatterplot(
  data = capitals
  , lat = "lat"
  , lon = "lon"
  , radius = 100000
  , fill_colour = "country"
  , layer_id = "scatter_layer"
)
