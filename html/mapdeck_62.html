mapdeck_legend R Documentation

Mapdeck Legend

Description

Constructs legend elements into the correct JSON format for plotting on the map

Usage

mapdeck_legend(legend_elements)

Arguments

legend_elements

vector of legend elements (made from legend_element)

See Also

legend_element

Examples

l1 <- legend_element(
	variables = c("a","b")
	, colours = c("#00FF00","#FF0000")
	, colour_type = "fill"
	, variable_type = "category"
	, title = "my title"
)

mapdeck_legend(l1)
