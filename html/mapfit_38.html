mmpp R Documentation

Create an MMPP

Description

Create an instance of MMPP (Markov-Modulated Poisson Process)

Usage

mmpp(size)

Arguments

size

An integer for the number of phases

Value

An instance of MMPP

Note

MMPP is a MAP whose D1 is given by a diagonal matrix.

Examples

## create an MMPP with 5 phases
(param1 <- mmpp(5))
