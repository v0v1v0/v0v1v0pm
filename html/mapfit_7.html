cf1 R Documentation

Create CF1

Description

Create an instance of CF1.

Usage

cf1(size, alpha, rate)

Arguments

size

An integer of the number of phases
alpha

A vector of initial probabilities
rate

A vector of rates

Value

An instance of CF1.

Examples

## create a CF1 with 5 phases
(param1 <- cf1(5))

## create a CF1 with 5 phases
(param1 <- cf1(size=5))

## create a CF1 with specific parameters
(param2 <- cf1(alpha=c(1,0,0), rate=c(1.0,2.0,3.0)))
