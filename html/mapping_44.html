popEU R Documentation

European population

Description

European population for year 2018

Usage

data("popEU")
data("popEUnuts2")

Format

A data frame with 2252 observations on the following 5 variables.

TIME

year

GEO

names

total

total

male

number of male

female

number of female

Source

https://ec.europa.eu/eurostat/data/database
