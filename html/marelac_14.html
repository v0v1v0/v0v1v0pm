convert_StoCl R Documentation

Salinity-Chlorinity Conversion

Description

Estimates the chlorinity concentration based on salinity

Usage

convert_StoCl(S = 35)

Arguments

S

salinity

Value

The chlorinity concentration, g/kg

Author(s)

Karline Soetaert <karline.soetaert@nioz.nl>

References
