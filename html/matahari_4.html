dance_save R Documentation

Save the log as an rds file.

Description

Save the log as an rds file.

Usage

dance_save(path)

Arguments

path

The path to the rds file.

Examples

## Not run: 

# Start recording
dance_start(value = TRUE)

# Execute some expressions
"Hello!"
4 + 4

# Stop recording
dance_stop()

# Save your log locally
dance_save("session.rds")

## End(Not run)
