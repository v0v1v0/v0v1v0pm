is.enum R Documentation

Is Object an Enum

Description

Test whether object has class Enum.

Usage

is.enum(x)

Arguments

x

object to be tested

Value

TRUE if x is an Enum, FALSE otherwise

Examples

HelloEnum <- Enum(
  "HelloEnum",
  Hello,
  World
)

# TRUE
is.enum(HelloEnum$Hello)

# FALSE
is.enum(5)
