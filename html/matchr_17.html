is.some R Documentation

Check if Option is Some

Description

Test whether Option Enum is Some or None.

Usage

is.some(x)

Arguments

x

object to be tested

Value

TRUE if x is enumerated type of variant Some, FALSE otherwise

Examples

x <- 1:5
get_n <- function(x, n) {
  if (n > length(x)) return(None)
  Some(x[n])
}
obj <- get_n(x, 6)
is.some(obj)  # FALSE
