fallthrough R Documentation

Fall Through Match

Description

Stop execution of current return expression in Match, then continue attempting to match conditions.

Usage

fallthrough()

Value

Object of class 'fallthrough'

Examples


Match(
  "abc",
  is.character -> {
    print("Found a character.")
    fallthrough()
  },
  "abc" -> "start of the alphabet",
  .     -> "found nothing"
)
