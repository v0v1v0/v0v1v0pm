Description

Add a name attribute to an element (most often, an R function)

Usage

name(x, name)

Arguments

x

an R object, e.g., an R function
name

the name of the object/function

Value

The object with the name attribute

Examples

f <- function(x) {sin(x)}
mathjax(call("integrate", name(f, "sin"), 0L, 2L*pi))
