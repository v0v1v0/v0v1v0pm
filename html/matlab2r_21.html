linspace R Documentation

Generate linearly-spaced vector

Description

This is a soft wrap around the base::seq() function.

Usage

linspace(x1, x2, n = 100L)

Arguments

x1

start point
x2

end point
n

length of output

Value

A numeric vector of n numbers between x1 and x2.

Author(s)

Waldir Leoncio

Examples

linspace(-5, 4)
linspace(1 + 2i, 9 + 9i, 5)
