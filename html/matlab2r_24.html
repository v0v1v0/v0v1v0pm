max R Documentation

Maximum (MATLAB version)

Description

Finds the minimum value for each column of a matrix, potentially returning the indices instead

Usage

max(X, indices = TRUE)

Arguments

X

matrix
indices

return indices?

Value

Either a list or a vector

Author(s)

Waldir Leoncio

Examples

A <- matrix(c(23, 42, 37, 15, 52))
max(A)
base::max(A) # for comparison
