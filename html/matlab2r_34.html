ones R Documentation

Matrix of ones

Description

wrapper of zeros_or_ones() that replicates the behavior of the ones() function on Matlab

Usage

ones(n1, n2 = n1, ...)

Arguments

n1

number of rows
n2

number of columns
...

extra dimensions

Value

An n1-by-n2 matrix of ones

Examples

ones(3)
ones(8, 1)
