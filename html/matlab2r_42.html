sortrows R Documentation

Sort rows of matrix or table

Description

Emulates the behavior of the sortrows function on Matlab

Usage

sortrows(A, column = 1)

Arguments

A

matrix
column

ordering column

Value

The A matrix sorted by the first row, then the second

Examples

mx <- matrix(c(3, 2, 2, 1, 1, 10, 0, pi), 4)
mx
sortrows(mx)
