disp R Documentation

Display the value of a variable

Description

disp(X) displays the value of variable X without printing the variable name. This is a wrapper around base::cat() that includes a breakline in the end.

Usage

disp(X)

Arguments

X

variable

Value

The value of X

Author(s)

Waldir Leoncio

Examples

A <- c(15, 150)
S <- 'Hello World.'
disp(A)
disp(S)
