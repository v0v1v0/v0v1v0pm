fix R Documentation

MATLAB fix function

Description

Rounds toward zero.

Usage

fix(A)

Arguments

A

numeric to be rounded

Details

Simply invokes trunc.

Value

Returns vector containing integers by truncating the corresponding values of argument A toward zero.

Author(s)

P. Roebuck proebuck1701@gmail.com

See Also

ceil, Round

Examples

fix(c(1.3, 2.5, 3.7))
