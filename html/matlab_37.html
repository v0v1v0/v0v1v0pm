numel R Documentation

MATLAB numel function

Description

Provides number of elements in array A or subscripted array expression.

Usage

numel(A, varargin)

Arguments

A

object of which to determine the number of elements
varargin

unimplemented

Value

Returns prod(size(A)).

Author(s)

P. Roebuck proebuck1701@gmail.com

See Also

prod, size

Examples

numel(2:9)	# 8
numel(magic(4))	# 16
