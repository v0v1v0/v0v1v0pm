rboolm R Documentation

Create matrix of random choosen boolean values

Description

Create matrix of random choosen boolean values

Usage

rboolm(nrow, ncol, true.proba = 0.5)

Arguments

nrow

number of rows
ncol

numer of columns
true.proba

probability of true values; default: 0.5

Value

a matrix

Examples

rboolm(3, 3)
rboolm(4, 5, true.proba = 0.3)
