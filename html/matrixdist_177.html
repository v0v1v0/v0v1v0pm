Description

Mean method for phase-type distributions

Usage

## S4 method for signature 'ph'
mean(x)

Arguments

x

An object of class ph.

Value

The raw first moment of the ph (or underlying ph) object.

Examples

set.seed(123)
obj <- ph(structure = "general", dimension = 3)
mean(obj)
