rMDPHstar R Documentation

Simulate MDPH*

Description

Generates a sample of size n from a MDPH* distribution with parameters alpha, S, and R.

Usage

rMDPHstar(n, alpha, S, R)

Arguments

n

Sample size.
alpha

Vector of initial probabilities.
S

Sub-transition matrix.
R

Reward matrix.

Value

Simulated sample.
