gridFifteenLeft R Documentation

Grid Fifteen Left

Description

This returns a fifteen grid left maze. These are standardized coordinates.

Usage

data(gridFifteenLeft)

Format

A data frame with 2 columns

start

Coordinates of Start Node.

end

Coordinates End Node.
