gridFourLeft R Documentation

Grid Four Left

Description

This returns a four grid left maze. These are standardized coordinates.

Usage

data(gridFourLeft)

Format

A data frame with 2 columns

start

Coordinates of Start Node.

end

Coordinates End Node.

Examples

# Returns a Grid with rank = 4
