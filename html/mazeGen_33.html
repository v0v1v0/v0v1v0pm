gridNineLeft R Documentation

Grid Nine Left

Description

This returns a nine grid left maze. These are standardized coordinates.

Usage

data(gridNineLeft)

Format

A data frame with 2 columns

start

Coordinates of Start Node.

end

Coordinates End Node.
