gridNineUp R Documentation

Grid Nine Up

Description

This returns a nine grid upwards maze. These are standardized coordinates.

Usage

data(gridNineUp)

Format

A data frame with 2 columns

start

Coordinates of Start Node.

end

Coordinates End Node.

Examples

## Not run: 

# Returns a Grid with rank = 9
data(gridNineUp)
