Grid Seven Right

Description

This returns a seven grid right maze. These are standardized coordinates.

Usage

data(gridSevenRight)

Format

A data frame with 2 columns

start

Coordinates of Start Node.

end

Coordinates End Node.

Examples

# Returns a Grid with rank = 7
