gridThreeLeft R Documentation

Grid Three Left

Description

This returns a three grid left maze. These are standardized coordinates.

Usage

data(gridThreeLeft)

Format

A data frame with 2 columns

start

Coordinates of Start Node.

end

Coordinates End Node.

Examples

## Not run: 

# Returns a Grid with rank = 3
data(gridThreeLeft)
