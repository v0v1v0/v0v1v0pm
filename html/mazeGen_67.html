gridThreeUp R Documentation

Grid Three Up

Description

This returns a three grid upwards maze. These are standardized coordinates.

Usage

data(gridThreeUp)

Format

A data frame with 2 columns

start

start, coordinates of Start Node.

carat

end, coordinates End Node.

Examples
