gridEightLeft R Documentation

Grid Eight Left

Description

This returns a eight grid left maze. These are standardized coordinates.

Usage

data(gridEightLeft)

Format

A data frame with 2 columns

start

Coordinates of Start Node.

end

Coordinates End Node.

Examples

## Not run: 

# Returns a Grid with rank = 8
data(gridEightLeft)
