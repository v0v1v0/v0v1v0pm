nRMSE R Documentation

Normalized root-mean-square error

Description

RMSE is normalized by the normalization constant

Usage

nRMSE(yhat, y, normConst)

Arguments

yhat

Model outputs
y

Observations
normConst

The normalization constant

Value

normalized RMSE value

Examples

x <- rnorm(100)
y <- rnorm(100)
nRMSE(x, y, sd(y))
