Reduction of Error

Description

Reduction of Error

Usage

RE(yhat, y, yc_bar)

Arguments

yhat

Model outputs in the validation set
y

Observations in the validation set
yc_bar

Mean observations in the calibration set

Value

RE value

Examples

x <- rnorm(100)
y <- rnorm(100)
yc_bar <- mean(x[1:50])
RE(x[51:100], y[51:100], yc_bar)
