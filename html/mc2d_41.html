Finite, Infinite, NA and NaN Numbers in mcnode.

Description

‘⁠is.na⁠’, ‘⁠is.nan⁠’, ‘⁠is.finite⁠’ and ‘⁠is.infinite⁠’ return a logical ‘⁠mcnode⁠’ of the same dimension as ‘⁠x⁠’.

Usage

## S3 method for class 'mcnode'
is.nan(x)
## S3 method for class 'mcnode'
is.finite(x)
## S3 method for class 'mcnode'

Arguments

A ‘⁠mcnode⁠’ object.

Value

A logical ‘⁠mcnode⁠’ object.
