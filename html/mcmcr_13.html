as.mcmcr R Documentation

Convert to an mcmcr object

Description

Converts an MCMC object to an mcmcr-object().

Usage

as.mcmcr(x, ...)

## S3 method for class 'mcarray'
as.mcmcr(x, name = "par", ...)

## S3 method for class 'mcmcarray'
as.mcmcr(x, name = "par", ...)

## S3 method for class 'nlist'
as.mcmcr(x, ...)

## S3 method for class 'nlists'
as.mcmcr(x, ...)

## S3 method for class 'mcmc'
as.mcmcr(x, ...)

## S3 method for class 'mcmc.list'
as.mcmcr(x, ...)

## S3 method for class 'mcmcrs'
as.mcmcr(x, ...)

Arguments

x

An MCMC object.
...

Unused.
name

A string specifying the parameter name.

Value

An mcmcr object.

Methods (by class)

  • mcarray: Convert an mcarray object to an mcmcr object

  • nlists: Convert an nlist::nlists-object() to an mcmcr object
