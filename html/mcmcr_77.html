is.mcmcarray R Documentation

Is mcmcarray object

Description

Tests whether an object is an mcmcarray-object().

Usage

is.mcmcarray(x)

Arguments

x

The object to test.

Value

A flag indicating whether the test was positive.

See Also

Other is: is.mcarray(), is.mcmcrs(), is.mcmcr()

Examples

is.mcmcarray(mcmcr_example$beta)
