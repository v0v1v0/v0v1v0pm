is.mcmcr R Documentation

Is mcmcr object

Description

Tests whether an object is an mcmcr-object().

Usage

is.mcmcr(x)

Arguments

x

The object to test.

Value

A flag indicating whether the test was positive.

See Also

Other is: is.mcarray(), is.mcmcarray(), is.mcmcrs()

Examples

is.mcmcr(mcmcr_example)
