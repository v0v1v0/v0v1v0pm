h_fmt_range R Documentation

Format and Concatenate to Range

Description

Help function to format numeric data as strings and concatenate into a single character range.

Usage

h_fmt_range(num1, num2, digits = c(2, 2), width = c(6, 6))

Arguments

num1

(numeric)
first numeric input.
num2

(numeric)
second numeric input.
digits

(integer)
the desired number of digits after the decimal point.
width

(integer)
the total field width.

Value

A single character.

See Also

h_fmt_num()

Examples

h_fmt_range(num1 = 3.14, num2 = 3.14, width = c(4, 4))
