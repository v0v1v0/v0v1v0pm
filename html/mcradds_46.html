printSummary R Documentation

Print Summary of a Regression Analysis

Description

A copy from mcr::printSummary in mcr package

Usage

printSummary(...)

Arguments

...

Arguments passed on to mcr::printSummary

.Object

object of type "MCResult".

Examples

data(platelet)
fit <- mcreg(
  x = platelet$Comparative, y = platelet$Candidate,
  method.reg = "Deming", method.ci = "jackknife"
)
printSummary(fit)
