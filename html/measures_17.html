KAPPA R Documentation

Cohen's kappa

Description

Defined as: 1 - (1 - p0) / (1 - pe). With: p0 = 'observed frequency of agreement' and pe = 'expected agremeent frequency under independence

Usage

KAPPA(truth, response)

truth

vector of true values
response
