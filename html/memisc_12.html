Negative Match

Description

%nin% is a convenience operator: x %nin% table is equivalent to !(x %in% table).

Usage

  x %nin% table

Arguments

x

the values to be matched
table

a values to be match against

Value

A logical vector

Examples

x <- sample(1:6,12,replace=TRUE)
x %in% 1:3
x %nin% 1:3
