rqm_to_sd R Documentation

rqm_to_sd

Description

Calculate the standard deviation from box plots

Usage

rqm_to_sd(min, LQ, UQ, max, n)

Arguments

min

Minimum value
LQ

Lower 75th quartile
UQ

Upper 75th quartile
max

Maximum value
n

Sample size

Value

Returns vector of standard deviation

Author(s)

Joel Pick

Examples

rqm_to_sd(min = 2, LQ = 3, UQ = 6, max = 9, n = 30)
