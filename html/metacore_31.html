xml_to_ds_vars R Documentation

XML to Data Set Var table

Description

Creates the ds_vars table, which acts as a key between the datasets and the var spec

Usage

xml_to_ds_vars(doc)

Arguments

doc

xml document

Value

data frame with the dataset and variables
