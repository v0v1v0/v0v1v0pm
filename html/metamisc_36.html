logit R Documentation

Apply logit tranformation

Description

Transforms values between 0 and 1 to values between -Inf and Inf.

Usage

logit(x)

Arguments

x

A vector of numerics (between 0 and 1)

Value

A vector of numerics (between -Inf and Inf).
