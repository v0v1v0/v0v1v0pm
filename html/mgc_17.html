gen.x.unif R Documentation

A helper function to generate n samples of a d-dimensional uniform vector.

Description

A helper function to generate n samples of a d-dimensional uniform vector.

Usage

gen.x.unif(n, d, a = -1, b = 1)

Arguments

n

the number of samples.
d

the number of dimensions.
a

the lower limit.
b

the upper limit.
x

[n, d] the simulated data matrix.
