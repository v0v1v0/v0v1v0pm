check_mhc_class R Documentation

Check the MHC class.

Description

Check the MHC class. Will stop if it is invalid.

Usage

check_mhc_class(mhc_class)

Arguments

mhc_class

MHC class. Must be I, II or NA. Use NA to let the class be deduced automatically

Value

Nothing.
