lbpmicox R Documentation

Survival data of 265 Low Back Pain Patients

Description

A data frame with 10 multiply imputed datasets of 265 observations each on 17 variables related to low back pain.

Usage

lbpmicox

Format

A data frame with 2650 observations on the following 18 variables.

Impnr

a numeric vector

patnr

a numeric vector

Status

dichotomous event

Time

continuous follow up time variable

Duration

continuous

Previous

dichotomous

Radiation

dichotomous

Onset

dichotomous

Age

continuous

Tampascale

continuous

Pain

continuous

Function

continuous

Satisfaction

categorical

JobControl

continuous

JobDemand

continuous

Social

continuous

Expectation

a numeric vector

Expect_cat

categorical
