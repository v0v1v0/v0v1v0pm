lbpmilr R Documentation

Data of 159 Low Back Pain Patients

Description

A data frame with 10 multiply imputed datasets of 159 observations each on 17 variables related to low back pain.

Usage

lbpmilr

Format

A data frame with 1590 observations on the following 17 variables.

Impnr

a numeric vector

ID

a numeric vector

Chronic

dichotomous

Gender

dichotomous

Carrying

categorical

Pain

continuous

Tampascale

continuous

Function

continuous

Radiation

dichotomous

Age

continuous

Smoking

dichotomous

Satisfaction

categorical

JobControl

continuous

JobDemands

continuous

SocialSupport

continuous

Duration

continuous

BMI

continuous
