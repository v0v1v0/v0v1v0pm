assignprobandkappas R Documentation

Computes probabilities of (relabeled) cluster and kappas.

Description

assignprobandkappas returns a list with information on probabilities of cluster belonging and Cohen's kappas.

Usage

assignprobandkappas(variables, k, metriccent, data, initialcluster)

Arguments

variables

internally provided by summary.miclust.
k

internally provided by summary.miclust.
metriccent

internally provided by summary.miclust.
data

internally provided by summary.miclust.
initialcluster

internally provided by summary.miclust.

Value

internal value to be used by summary.miclust.
