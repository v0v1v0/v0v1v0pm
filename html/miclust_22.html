relabelclusters R Documentation

Relabel clusters.

Description

relabelclusters relabels the clusters so that they all have the same meaning in the all the data sets.

Usage

relabelclusters(refcluster, cluster)

Arguments

refcluster

internally provided by assignprobandkappas function.
cluster

internally provided by assignprobandkappas function.

Value

internal value to be used by assignprobandkappas function.
