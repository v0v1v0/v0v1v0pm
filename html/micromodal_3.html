use_micromodal R Documentation

Use micromodal

Description

Call this function once in your app's UI.

Usage

use_micromodal()

Details

This function adds the dependencies needed for the modals.

See micromodal() for a complete example.

Value

htmltools::tagList()

Examples

if (interactive()) {
  library(shiny)
  library(micromodal)

  ui <- fluidPage(
    use_micromodal(),
    # the rest of your UI code
  )
}
