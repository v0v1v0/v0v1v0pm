cv.kappa R Documentation

Critical Values

Description

Computes critical values for the kappa_test statistic

Usage

cv.kappa(t, alpha, sig.lev)

Arguments

t

Sample size.
alpha

Value of the tail index between 2 and 4.
sig.lev

Significance level.

Value

Critical value.
