str_rev R Documentation

Reverse Text Strings

Description

Create a reverse version of strings.

Usage

str_rev(x)

strrev(x)

Arguments

x

vector with strings. Is coerced to character.

Value

Returns a character vector with reversed strings.

Author(s)

Sven E. Templer

See Also

rev

Examples

#

s <- c("abc","asdf")

str_rev(s)
