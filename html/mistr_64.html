fdist R Documentation

Creates an Object Representing F Distribution

Description

The function creates an object which represents the F distribution.

Usage

fdist(df1 = 2, df2 = 2)

Arguments

df1

degrees of freedom parameter, default: 2.
df2

degrees of freedom parameter, default: 2.

Details

See FDist.

Value

Object of class fdist.

See Also

FDist

Examples

f <- fdist(2, 2)
d(f, c(2, 3, 4, NA))
r(f, 5)
