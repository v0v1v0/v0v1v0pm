autolayout R Documentation

Automatic layout for several plots in one figure

Description

Returns a matrix which can be used in layout function as its mat argument.

Usage

autolayout(np)

Arguments

np

number of plots that are to be produced on 1 figure

Value

A matrix.

Author(s)
