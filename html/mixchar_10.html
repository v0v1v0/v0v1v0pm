model_fit R Documentation

Accessor function to extract model fit

Description

Accessor function to extract model fit

Usage

model_fit(object)

Arguments

object

a decon object

Value

$minpack.lm of the object

Examples


data(juncus)
tmp <- process(juncus, init_mass = 18.96,
               temp = 'temp_C', mass_loss = 'mass_loss')
output <- deconvolve(tmp)
model_fit(output)
