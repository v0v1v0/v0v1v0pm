rate_data R Documentation

Accessor function to extract processed dataframe

Description

Accessor function to extract processed dataframe

Usage

rate_data(object)

Arguments

object

a process or deconvolve object

Value

Dataframe of the object

Examples

data(juncus)
tmp <- process(juncus, init_mass = 18.96,
               temp = 'temp_C', mass_loss = 'mass_loss')
rate_data(tmp)
