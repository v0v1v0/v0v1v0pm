validParams R Documentation

Validate MizerParams object and upgrade if necessary

Description

Checks that the given MizerParams object is valid and upgrades it if necessary by calling upgradeParams().

Usage

validParams(params)

Arguments

params

The MizerParams object to validate

Details

Besides upgrading if necessary, the only change that may be made to the given MizerParams object is that the w_min_idx slot is recalculated.

Value

A valid MizerParams object
