LearnerGlm R6 class

Description

This learner is a wrapper around stats::glm() in order to perform a generalized linear regression. There is no implementation for tuning parameters.

Details

Can be used with

  • MLCrossValidation

Implemented methods:

  • ⁠$predict⁠ To predict new data with the model.
