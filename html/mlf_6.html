get_mse R Documentation

Mean Squared Error

Description

Estimates mean squared error from model predictions.

Usage

get_mse(truth, estimate)

Arguments

truth

test data vector or baseline accuracy to test against.
estimate

predicted vector

Examples

# Sample data
test <- rnorm(25, 80, 35)
predicted <- rnorm(25, 80, 50)

mlf::get_mse(test, predicted)
