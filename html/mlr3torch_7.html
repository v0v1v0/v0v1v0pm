as_torch_callback R Documentation

Convert to a TorchCallback

Description

Converts an object to a TorchCallback.

Usage

as_torch_callback(x, clone = FALSE, ...)

Arguments

x

(any)
Object to be converted.
clone

(logical(1))
Whether to make a deep clone.
...

(any)
Additional arguments

Value

TorchCallback.
