is.nullcpo R Documentation

Check for NULLCPO

Description

Check whether the given object is a NULLCPO.

Usage

is.nullcpo(x)

Arguments

x

[any]
The object to check

Value

[logical(1)]. TRUE if x is a NULLCPO, FALSE otherwise.
