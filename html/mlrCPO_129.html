is.retrafo R Documentation

Check CPORetrafo

Description

Check whether the given object is a CPORetrafo object.

Usage

is.retrafo(x)

Arguments

x

[any]
The object to check.

Value

TRUE if x has class CPORetrafo, FALSE otherwise.
