nullcpoToNull R Documentation

NULLCPO to NULL

Description

Convert NULLCPO to NULL, leave other values intact.

Usage

nullcpoToNull(cpo)

Arguments

cpo

[CPO]
The cpo.

Value

[CPO | NULL]. NULL if cpo is NULLCPO, cpo otherwise.

See Also

Other NULLCPO related: NULLCPO, is.nullcpo(), nullToNullcpo()
