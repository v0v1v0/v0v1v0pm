nullToNullcpo R Documentation

NULL to NULLCPO

Description

Convert NULL to NULLCPO, leave other values intact.

Usage

nullToNullcpo(cpo)

Arguments

cpo

[CPO | NULL]
The CPO.

Value

[CPO]. NULLCPO if cpo is NULL, cpo otherwise.

See Also

Other NULLCPO related: NULLCPO, is.nullcpo(), nullcpoToNull()
