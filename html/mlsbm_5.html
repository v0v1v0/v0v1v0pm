mean_CRI R Documentation

The mean_CRI function

Description

Simple function to return the mean (95% CrI) for a vector

Usage

mean_CRI(y, dig = 2)

Arguments

y

A numeric vector
dig

The number of digits to round to

Value

A string of mean and 95% quantile interval rounded to 'dig'

Examples

mean_CRI(rnorm(1000))
