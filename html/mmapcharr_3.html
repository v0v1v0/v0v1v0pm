dim_file R Documentation

File dimensions

Description

Number of lines and columns of file (and extra 'return' characters).

Usage

dim_file(file)

Arguments

file

Path to file.

Value

The number of lines and columns of file (and extra 'return' characters).

Examples

tmpfile <- tempfile()
write(0:9, tmpfile, ncolumns = 2)
dim_file(tmpfile)
