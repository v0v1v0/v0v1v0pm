cartesianExpand R Documentation

expands two data.frames using the Cartesian product

Description

takes the cartesian product of two data.frames

Usage

cartesianExpand(x, y)

Arguments

x

a data.frame
y

a data.frame

Value

a data.frame

Examples

x = data.frame("a" = 1:5, "b" = 6:10)
y = data.frame("z" = letters[1:5], "y" = letters[6:10])
cartesianExpand(x, y)
