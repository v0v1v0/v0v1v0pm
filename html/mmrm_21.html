h_df_1d_kr R Documentation

Calculation of Kenward-Roger Degrees of Freedom for One-Dimensional Contrast

Description

Calculates the estimate, adjusted standard error, Kenward-Roger degree of freedom, t statistic and p-value for one-dimensional contrast. Used in df_1d() if method is "Kenward-Roger" or "Kenward-Roger-Linear".

Usage

h_df_1d_kr(object, contrast, linear = FALSE)

Arguments

object

(mmrm)
object created by mmrm() with Kenward-Roger(-Linear) method.
contrast

(numeric)
contrast vector.
linear

(logical)
whether to use linear Kenward-Roger approximation.

Value

List with est, se, df, t_stat and p_val.
