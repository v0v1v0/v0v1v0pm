h_df_1d_list R Documentation

Creating Results List for One-Dimensional Contrast

Description

Creates a list of results for one-dimensional contrasts using the estimated values from quadratic form.

Usage

h_df_1d_list(est, var, v_num, v_denom)

Arguments

est

(number)
estimate.
var

(number)
variance of estimate.
v_num

(number)
numerator for Satterthwaite d.f.
v_denom

(number)
denominator for Satterthwaite d.f.

Value

List with est, se, df, t_stat and p_val (2-sided p-value).
