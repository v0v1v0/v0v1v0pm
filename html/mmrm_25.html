h_df_md_kr R Documentation

Calculation of Kenward-Roger Degrees of Freedom for Multi-Dimensional Contrast

Description

Calculates the Kenward-Roger degree of freedom, F statistic and p value for multi-dimensional contrast. Used in df_md() if method is "Kenward-Roger" or "Kenward-Roger-Linear".

Usage

h_df_md_kr(object, contrast, linear = FALSE)

Arguments

object

(mmrm)
object.
contrast

(matrix)
contrast matrix.
linear

(logical)
whether to use linear Kenward-Roger approximation.

Value

List with num_df, denom_df, f_stat and p_val (2-sided p-value).
