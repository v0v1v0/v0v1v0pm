h_df_md_list R Documentation

Creating Results List for Multi-Dimensional Contrast

Description

Calculate the p-value using the F statistic and the numerator/denominator degree of freedom and return a list.

Usage

h_df_md_list(f_stat, num_df, denom_df)

Arguments

f_stat

(number)
F-statistic.
num_df

(number)
numerator degrees of freedom.
denom_df

(number)
denominator degrees of freedom.

Value

List with num_df, denom_df, f_stat and p_val (2-sided p-value).
