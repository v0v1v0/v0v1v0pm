h_jac_col_as_matrix R Documentation

Formatting a Column from Jacobian Matrix as Matrix

Description

Convert a column of Jacobian matrix into square matrix.

Usage

h_jac_col_as_matrix(jac_matrix, col)

Arguments

jac_matrix

(matrix)
full Jacobian matrix.
col

(int)
column index.

Value

The column col of jac_matrix as a square matrix. Here the values in the column are used to fill the result column by column.
