date_tidy R Documentation

A function that makes date variables returned from the API datable, ie by converting them to POSIXct. Does the same thing for datetimes.

Description

A function that makes date variables returned from the API datable, ie by converting them to POSIXct. Does the same thing for datetimes.

Usage

date_tidy(df)

Arguments

df

The tibble with the undateable dates.
