RowFactor R Documentation

Use a variable as a factor to give rows in a table.

Description

Use a variable as a factor to give rows in a table.

Usage

RowFactor(
  x,
  name = deparse(expr),
  levelnames = levels(x),
  spacing = 3,
  space = 1,
  suppressfirst = TRUE,
  nopagebreak = "\\nopagebreak ",
  texify = getOption("tables.texify", FALSE),
  expr = substitute(x),
  override = TRUE
)
