ER.normal R Documentation

Utility for ER function

Description

The ER function is an implementation of the ER-algorithm of Little and Smith (1987).

Usage

ER.normal(
  data,
  weights = rep(1, nrow(data)),
  psi.par = c(2, 1.25),
  np = sum(weights),
  p = ncol(data),
  s.counts,
  s.id,
  S,
  missing.items,
  nb.missing.items,
  start.mean = rep(0, p),
  start.var = diag(1, p),
  numb.it = 10,
  Estep.output = FALSE,
  tolerance = 1e-06
)

Arguments

data

matrix or dataframe with data.
weights

vector of weights.
psi.par

parameters for psi function.
np

np.
p

number of columns.

s.counts.

s.id.
S

S.

missing items.

number of iterations.
Estep.output

Estep.output.

tolerance.
