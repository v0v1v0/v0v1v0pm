skewness R Documentation

Skewness of the sample

Description

This function computes skewness of given data.

Usage

skewness(x, na.rm = FALSE)

Arguments

x

a numeric vector, matrix or data frame.

na.rm

logical. Should missing values be removed?

Author(s)

Lukasz Komsta

See Also

agostino.test

Examples

set.seed(1234)
skewness(rnorm(1000))
