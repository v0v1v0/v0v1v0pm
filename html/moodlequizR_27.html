sa R Documentation

sa

Description

This function creates a text question for moodle in CLOZE format.

Usage

sa(txt, w = 100, caps = TRUE, pts = 1)

Arguments

txt

character vector with possible answers
w

vector of weights
caps

keep capital letters
pts

points for answers

Value

a character vector

Examples

sa("Los Angeles")
sa(c("Los Angeles", "San Francisco"), w=c(100, 80))
