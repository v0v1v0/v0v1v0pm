rates R Documentation

Mopac Express Lane Rates

Description

Express Lane Rates for Loop 1 "Mopac" in Austin, Texas. Rates correspond to RM 2222 / Far West entry points with pre-COVID weekday traffic. Recommend loading readr package so that time displays correctly as hms.

Usage

rates

Format

A tibble with 25 rows and 3 variables:

direction

north: Far West entry point heading to Parmer, south: RM 2222 heading to Cesar Chavez

time

time of day

rate

TxTag rate in USD

Source

Rates obtained from Central Texas Regional Mobility Authority. https://www.mobilityauthority.com/pay-your-toll/rates
