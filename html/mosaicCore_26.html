logit R Documentation

Logit and inverse logit functions

Description

Logit and inverse logit functions

Usage

logit(x)

ilogit(x)

Arguments

x

a numeric vector

Value

For logit the value is

log(x/(1 - x))

For ilogit the value is

exp(x)/(1 + exp(x))

Examples

p <- seq(.1, .9, by=.10)
l <- logit(p); l
ilogit(l)
ilogit(l) == p
