modelVars R Documentation

extract predictor variables from a model

Description

extract predictor variables from a model

Usage

modelVars(model)

Arguments

model

a model, typically of class lm or glm

Value

a vector of variable names

Examples

if (require(mosaicData)) {
  model <- lm( wage ~ poly(exper, degree = 2), data = CPS85 )
  modelVars(model)
}
