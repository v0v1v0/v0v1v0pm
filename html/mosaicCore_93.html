rhs_or_expr R Documentation

Return rhs of a formula or expression

Description

Return rhs of a formula or expression

Usage

rhs_or_expr(x)

Arguments

x

A formula or some other object to be quoted

Examples

# This should evaluate to TRUE
rhs_or_expr(~z)
rhs_or_expr(z)
identical(rhs_or_expr(~z), rhs_or_expr(z))
