Description

Get filtered views

Usage

get_filtered_views(r, data_points, n, p, filetype = "rda")

Arguments

r

Recording object.
data_points

vector of body parts e.g. 'Nose'.
n

window size.
p

poly order.
filetype

type of file ('rda' as default), can be 'csv'.

Value

list of FilteredView objects.
