get_joined_view R Documentation

Get joined view from multiple views from the same recording

Description

Get joined view from multiple views from the same recording

Usage

get_joined_view(l, folder_out = "Joined", save_output = FALSE)

Arguments

l

named list of View objects.
folder_out

output folder relative to recording home (default is 'Joined').
save_output

save the output?

Value

JoinedView object

Examples

r <- get_sample_recording()
rv_list <- get_raw_views(r)
jv <- get_joined_view(rv_list)
