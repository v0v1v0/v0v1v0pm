round.mpoly R Documentation

Round the coefficients of a polynomial

Description

Round the coefficients of an mpoly object.

Usage

## S3 method for class 'mpoly'
round(x, digits = 3)

Arguments

x

an mpoly object
digits

number of digits to round to

Value

the rounded mpoly object

Author(s)

David Kahle david@kahle.io

See Also

mp()

Examples


p <- mp("x + 3.14159265")^4
p
round(p)
round(p, 0)

n <- 101

# one dimensional case
df <- data.frame(x = s)
df <- mutate(df, y = -x^2 + 2*x - 3 + rnorm(n, 0, 2))
qplot(x, y, data = df)
mod <- lm(y ~ x + I(x^2), data = df)
p <- as.mpoly(mod)
qplot(x, y, data = df) +
  stat_function(fun = as.function(p), colour = 'red')

p
round(p, 1)
qplot(x, y, data = df) +
  stat_function(fun = as.function(p), colour = 'red') +
  stat_function(fun = as.function(round(p,1)), colour = 'blue')
