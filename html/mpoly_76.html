vars R Documentation

Determine the variables in a mpoly object.

Description

Determine the variables in a mpoly object.

Usage

vars(p)

Arguments

p

An mpoly or mpolyList object.

Value

A character vector of the variable names.

Examples


p <- mp("x + y^2")
vars(p)

p <- mp(c("x + y^2", "y - 2 x"))
vars(p)
