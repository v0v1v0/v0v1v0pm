is_url R Documentation

Checks whether the variable is a valid URL

Description

Checks whether the variable is a valid URL

Usage

is_url(x)

Arguments

x

(any) The object to test

Value

TRUE if x is a valid URL, FALSE otherwise

Examples

is_url("http://something.com")
is_url("https://google.com")

is_url(1)
is_url("foo")
is_url(NA)
