match_ij R Documentation

Fast Match for Matrix Indexes

Description

Match ix,jx-couple in ti,tj-table and return their 1-based positions (0 for non matched couples)

Usage

match_ij(ix, jx, ti, tj)

Arguments

ix

An integer vector
jx

An integer vector
ti

An integer vector
tj

An integer vector

Value

An integer vector

Examples

match_ij(1:2, 1:2, 0:4, 0:4)
# [1] 2 3
