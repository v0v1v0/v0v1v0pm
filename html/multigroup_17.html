plot.mg R Documentation

Plots for multigroup objects

Description

plots of variables (loadings) and individuals (scores) if TRUE

Usage

## S3 method for class 'mg'
plot(x, axes = c(1, 2), cex = NULL, font.lab = NULL, ...)

Arguments

x

results of multigroup method in the package
axes

by default the first two components
cex

character expansion for text by default .85
font.lab

type of font by default 3
...

Further arguments are ignored

Value

loadings and scores plots
