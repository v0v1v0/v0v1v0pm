remove_jdk R Documentation

Remove Jdk

Description

Uninstall jdk from this package.

Usage

remove_jdk(path = jdk_path())

uninstall_jdk(path = jdk_path())

Arguments

path

The path in which jdk is installed.

See Also

Other jdk-installer: install_jdk(), jdk_path()

Other jdk-installer: install_jdk(), jdk_path()

Examples

## Not run: 
  remove_jdk()
  uninstall_jdk()

## End(Not run)
