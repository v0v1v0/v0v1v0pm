Number of Outcomes in a Multimedia Object

Description

Number of Outcomes in a Multimedia Object

Usage

n_outcomes(object)

Arguments

object

An object of class multimedia.

Value

An integer specifying the number of outcomes.

Examples

exper <- demo_joy() |>
    mediation_data("PHQ", "treatment", starts_with("ASV"))
multimedia(exper) |>
    n_outcomes()
