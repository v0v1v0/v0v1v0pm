write.dat R Documentation

Write dat Files

Description

A function to write dat files.

Usage

write.dat(x, path)

Arguments

x

an object representing the multiple network structure

path

the path file for the output

Details

“dat” files are the format used in the Pacnet program inside StOCNET. In case that the input data represents a multiple network then a separate file will be produced, each one representing a single type of relationship in the system. The name of the output files depends on the object title.

Value

File(s) output with adjacency matrices with a .dat format.

Note

In case that the directory in the path for the output does not exist then a folder will be created automatically.
