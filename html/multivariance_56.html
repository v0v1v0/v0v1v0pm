pearson.pvalue.unif R Documentation

compute the p-value by Pearson's approximation assuming uniform marginals and euclidean distance

Description

compute the p-value by Pearson's approximation assuming uniform marginals and euclidean distance

Usage

pearson.pvalue.unif(
  x,
  vec = NA,
  type = "total",
  psi = NULL,
  isotropic = TRUE,
  ...
)

Examples

multivariance:::pearson.pvalue.unif(matrix(runif(300),ncol = 3))
