f2009 R Documentation

2009 FBS College Football Regular Season Data

Description

2009 FBS College Football Regular Season Data

Usage

data(f2009)

Format

A data frame with 772 observations on the following 7 variables.

home

a factor

Game.Date

a POSIXlt date variable

away

a factor

home.response

a numeric vector

home.score

a numeric vector

away.response

a numeric vector

away.score

a numeric vector

neutral.site

a numeric vector

partition

a numeric vector

Source

http://web1.ncaa.org/mfb/download.jsp?year=2009&div=IA

Examples

data(f2009)
