f2010 R Documentation

2010 FBS College Football Regular Season Data

Description

2010 FBS College Football Regular Season Data

Usage

data(f2010)

Format

A data frame with 770 observations on the following 9 variables.

home

a factor

Game.Date

a POSIXlt

away

a factor

home.response

a numeric vector

home.score

a numeric vector

away.response

a numeric vector

away.score

a numeric vector

neutral.site

a numeric vector

partition

a numeric vector

Source

http://web1.ncaa.org/mfb/download.jsp?year=2010&div=IA

Examples

data(f2010)
