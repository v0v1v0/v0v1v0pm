2012 FBS College Football Regular Season Data

Description

2012 FBS College Football Regular Season Data

Usage

data(f2012)

Format

A data frame with 809 observations on the following 9 variables.

home

a factor

Game.Date

a POSIXlt

away

a factor

home.response

a numeric vector

home.score

a numeric vector

away.response

a numeric vector

away.score

a numeric vector

neutral.site

a numeric vector

partition

a numeric vector

Source

http://web1.ncaa.org/mfb/download.jsp?year=2012&div=IA
