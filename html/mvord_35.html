pseudo_R_squared R Documentation

Pseudo R^2 for objects of class 'mvord'

Description

This function computes Mc Fadden's Pseudo R^2 for objects of class 'mvord'.

Usage

pseudo_R_squared(object, adjusted = FALSE)

Arguments

object

an object of class 'mvord'.
adjusted

if TRUE, then adjusted Mc Fadden's Pseudo R^2 is computed.
