ooom R Documentation

One over one minus a multivariate polynomial

Description

Uses Taylor's theorem to give one over one minus a multipol

Usage

ooom(P,n)

Arguments

n

Order of expansion
P

Multivariate polynomial

Author(s)

Robin K. S. Hankin

See Also

horner

Examples

ooom("x",5)
ooom("x",5) * as.mvp("1-x")  # 1 + O(x^6)


ooom("x+y",4)

"x+y" %>% ooom(5) %>% `-`(1) %>% ooom(3)
