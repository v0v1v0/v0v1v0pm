wis_lakes R Documentation

Example of lake ID data from Wisconsin

Description

This data.frame contains the names and lake IDs for 10 lakes in Wisconsin. These are used in examples throughout the package.

Usage

wis_lakes

Format

wis_lakes

A data.frame with 10 rows and 4 columns

state

The state abbreviation (i.e. "wi")

county

The name of the county in which the lake is found

lake.id

A unique identifier for each lake

lake.name

The name of the lake
